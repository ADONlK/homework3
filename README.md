#homework3
реализован шаблонный класс Heap<T, SIZE>, представляющий собой максимальную кучу фиксированного размера.
Куча хранит элементы в массиве и поддерживает следующие операции:
добавление элемента (push),
удаление и получение максимального элемента (pop),
получение текущего максимального элемента (top),
проверка на пустоту и заполненность (empty, full),
получение текущего количества элементов (size).
Класс является шаблонным и может работать с любыми типами данных, для которых определён оператор сравнения <.
Тестируем в main.cpp
CLion работает через CMake, поэтому CMakeLists, а не Makefilе
